Tendencias en la Ingeniería del Software

Desde sus inicios hasta la actualidad

1950s

Primeros pasos

Surgen las primeras computadoras electrónicas. La programación se hacía en lenguaje máquina y ensamblador. Aunque rudimentarios, estos avances sentaron las bases de la informática y del concepto de software.

UNIVAC Lenguaje Máquina Ensamblador
1960s

Lenguajes de alto nivel y Crisis del Software

FORTRAN, COBOL y ALGOL impulsan el desarrollo de software. Surge la "crisis del software", al evidenciar fallos en proyectos grandes y costosos. Se hace evidente la necesidad de metodologías más formales.

FORTRAN COBOL ALGOL IBM System/360
1970s

Nacimiento de la Ingeniería de Software

Se introduce el Modelo en Cascada (Royce, 1970). Lenguajes como C (1972) marcan un antes y un después. Se consolida la noción de ciclo de vida del software y de procesos organizados.

Modelo Cascada Lenguaje C UNIX Pascal
1980s

Orientación a Objetos

Con C++ y Smalltalk surge la Programación Orientada a Objetos (POO), que introduce modularidad y reutilización. También aparecen interfaces gráficas, acercando la informática a usuarios no técnicos.

C++ Smalltalk POO GUI
1990s

Globalización y la Web

Internet cambia la forma de trabajar. Surgen lenguajes como Java, PHP y JavaScript (1995). Se impulsa el software distribuido y comienzan a aparecer nuevas prácticas frente al modelo en cascada.

Java JavaScript PHP World Wide Web
2000s

Metodologías Ágiles y Software Libre

El Manifiesto Ágil (2001) revoluciona el desarrollo con Scrum y XP. El software libre (Linux, Apache, MySQL) se expande, generando comunidades activas e impulsando la colaboración global.

Scrum XP Linux Open Source
2010s

Movilidad, Cloud y DevOps

Las apps móviles dominan el mercado. El Cloud Computing (AWS, Azure, Google Cloud) transforma la infraestructura. DevOps automatiza despliegues e integra desarrollo con operaciones.

iOS/Android AWS DevOps Docker
2020s

Inteligencia Artificial y Microservicios

La IA y el Machine Learning se integran en casi todo el software moderno. Los microservicios y serverless dominan la arquitectura. La ciberseguridad se convierte en eje central de la industria.

AI/ML Microservicios Serverless Kubernetes
